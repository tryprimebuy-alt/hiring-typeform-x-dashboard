## Scoring System

### How Scores Work
Each scored question has:
- **Weight** (0вҖ“10): How important this criterion is
- **Raw Score**: Points earned for the answer
- **Weighted Score**: `rawScore Г— (weight / 10)`
- **Final %**: `sum(weighted) / sum(maxWeighted) Г— 100`

### Tiers
| Tier | Range | Action |
|------|-------|--------|
| рҹҹў High | 70%+ | Schedule interview |
| рҹҹЎ Mid | 40вҖ“69% | Review manually |
| рҹ”ҙ Low | <40% | Auto-reject or hold |

### Textarea Scoring (Heuristic)
For open-ended responses, a length-based heuristic is used by default:
- 50 chars вҶ’ score 2
- 100 chars вҶ’ score 5
- 200+ chars вҶ’ score 10 (capped)

> **Note:** Replace with manual scoring or AI-based scoring when Claude Code integration is ready.
